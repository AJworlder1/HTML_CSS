В предыдущих версиях CSS разработчики были вынуждены использовать только те шрифты, которые гарантированно установлены на компьютере пользователыя, CSS3 разработчики могут использовать любые шрифты, которые они захотят. Когда Вы найдете необходимый шрифт, просто разместите его на веб-сервере и подлючите его с помощью нового CSS3 свойства @font-face. Подключённый шрифт будет скачан и отображён автоматически на загрузке страницы.

Обратите внимание:Internet Exploler поддерживает шрифты в формате .eot, а остальные браузеры в формате .ttg т.е. для того, чтобы щрифт отображался корректно во всех браузерах, Вы должны подключить его к странице в двух форматах
